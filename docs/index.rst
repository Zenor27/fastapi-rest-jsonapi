Welcome to FastAPI REST JSONAPI's documentation!
================================================

FastAPI REST JSONAPI is an extension for FastAPI.

It provides a simple, yet powerful, way to build REST APIs around the `JSONAPI specification <https://jsonapi.org/>`__.


Compatibility
=============

FastAPI REST JSONAPI requires Python 3.9+
